이름을 입력받아 위치값 알아내기

indexOf(item, start) : 배열에서 요소를 찾아 위치를 리턴한다.

lastIndexOf(item, start) : 역순으로 요소를 찾아 위치를 리턴한다.

push(a,b,c,...) : 배열 끝에 요소룰 추가한다.

pop() : 마지막 요소를 제거하고 리턴한다.

sort 앞자리 수 기준으로 정렬

sort ascending 오름차순 정렬

ascending : (오름차순) : 작은 수 -> 큰 수

sort descending 내림차순 정렬

descending : (내림차순) : 큰 수 -> 작은 수

slice배열 요소 자르기
