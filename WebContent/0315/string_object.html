str = "우리 나라 좋은 나라 대한 민국";

charAt(idx) : idx번째 문자를 추출한다.

charCodeAt(idx) : idx번째 문자의 Unicode값을 구한다.

indexOf("문자") : "문자"가 앞에서부터 몇번째 index에 있는지 찾아서 출력

lastIndexOf("문자") : "문자"가 뒤에서부터 찾고 앞에서 몇번째 index에 있는지 찾아서 출력


substring(start, end) : start에서부터 end-1까지의 문자열을 추출


slice(start, end) : start에서부터 end-1까지의 문자열을 자른다


substr(start, length): start에서부터 length갯수만큼 문자를 추출.


split(separator, limit): separator(구분자)를 기준으로 해서 문자열을 분리, 분리되는 문자 갯수
